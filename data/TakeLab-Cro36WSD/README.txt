1 Description

Cro36WSD is a medium-scale WSD lexical sample for Croatian which also adapts the
multi-label annotation scheme. Cro36WSD comprises multi- and single-label dataset variants.</p>
The construction of the dataset is described in:

  Domagoj Alagić and Jan Šnajder (2015). Cro36WSD: A Lexical Sample for Croatian Word Sense Disambiguation. Proceedings of the 10th edition 
  of the Language Resources and Evaluation Conference (LREC2016). In press.

If you use this dataset for your own work, please cite the above paper.

2 Dataset

The dataset is available from here: TakeLab-Cro36WSD.tar.gz.

The archive contains three folders:

- one containing the sense inventory we constructed (/sense-inventory)
- one containing the annotation data from five annotators (/annotation_data)
- one containing all the dataset variants (/datasets):
--- Cro36WSD-M: a multi-label variant of the dataset (/datasets/Cro36WSD-M)
--- Cro36WSD-S: a single-label variant of the dataset (/datasets/Cro36WSD-S)
